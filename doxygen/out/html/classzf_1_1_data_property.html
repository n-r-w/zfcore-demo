Detailed Description

Свойство.

Для оптимизации скорости доступа, данные связанные со свойствами, хранятся в векторах, размер которых определяется максимальным id свойств данной структуры. Поэтому желательно в рамках одной структуры задавать непрерывные номера id начиная с 0 или 1

Правило автоматического обновления значения свойства на основании другого свойства Сейчас поддерживается только обновление из lookup таблиц.

Пример: В нашей модели есть поле ID_USER, где хранится id пользователя и поле USER_NAME, куда надо автоматически поместить имя пользователя. Устанавливаем setDataSource для поля USER_NAME. При этом мы связываем наше поле ID_USER с моделью списка пользователей и указываем его набор данных и колонку, отвечающую за имя пользователя. После этого любое изменение ID_USER приведет к заполнению поля USER_NAME Добавляет PropertyLink типа PropertyLinkType::DataSource
