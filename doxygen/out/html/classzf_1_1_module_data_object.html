Уникальный ключ строки набора данных.

Запрашивает значение из getDatasetRowID и если его нет, то формирует уникальный идентификатор, который затем записывается в методе writeDatasetRowID. Запись производится прямо в этом методе, с временным отключением генерации сигналов ItemModel. RowID хранится в самой первой колонке в роли Consts::UNIQUE_ROW_COLUMN_ROLE. В момент добавления новой строки, в ней нет данных. Следовательно даже при наличии колонки с признаком PropertyOption::Id, невозможно сформировать уникальный идентификатор на основе реального ID. Поэтому при добавлении строк, генерируется уникальный идентификатор, не привязанный к данным через метод RowID::createFull
