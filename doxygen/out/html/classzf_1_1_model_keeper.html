Detailed Description

Класс предназначен для централизованного хранения списков моделей, чтобы они не сбрасывались когда счетчик указателя падает до нуля.

При каждом запросе keepModels, загруженные модели сохраняются и хранятся до тех пор, пока объект не будет удален или не будет вызов freeModels
