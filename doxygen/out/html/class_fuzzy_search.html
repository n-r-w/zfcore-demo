Detailed Description

Алгоритм нечеткого поиска Для сравнения двух строк на "похожесть" используется модифициованный метод "Расстояние Дамерау-Левенштейна" с нормализацией результата.

В метод вычисления "расстояния" между строками добавлен учет похожести звучания отдельных букв т.е. стоимость замены 'о' на 'я' будет выше чем замена 'о' на 'а'
