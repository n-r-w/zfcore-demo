Получить объект по его идентификатору Если объект используется, то сразу возвращается указатель.

Если есть в кэше, то копируется из кэша. Иначе создается новый

Parameters
is_detachedОбъект должен быть отсоединен. Т.е. если такой уже используется, то надо сделать его копию. Если не используется - загрузить
is_from_cacheВзято из кэша
is_clonedКлонировано. Данные клонируются при условии:
  • Запрашивается отсоединенный объект
  • Объект с таким идентификатором уже используется
is_newСоздан новый объект
errorЕсли ошибка, то метод возвращает nullptr
