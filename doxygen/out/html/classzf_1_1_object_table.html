Detailed Description

template<typename Entity, typename Property>
class zf::ObjectTable< Entity, Property >

Шаблонный класс для таблиц свойств Для классов, участвующих в шаблоне должна быть определена функция QString zfUniqueKey(const T& value); которая должна вернуть уникальный ключ для значения
