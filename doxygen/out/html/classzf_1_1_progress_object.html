Detailed Description

Базовый класс отслеживающий состояние прогресса Прогресс имеет стек вызовов.

Каждый вызов метода startProgress увеличивает уровень вложенности прогресса. Все методы работы с процентом или текстом прогресса относятся к нижнему уровню стека

◆ setLongOperation()

void zf::ProgressObject::setLongOperation ( ) const

Отмечает данную операцию как длительную.

Имеет смысл для операций, для которых нет возможности установки промежуточных процентов между 0 и 100 (например запрос к БД). Если операци отмечена как длительная, то она будет принудительно отображаться в окне прогресса. Сбрасывается в false при окончании прогресса
