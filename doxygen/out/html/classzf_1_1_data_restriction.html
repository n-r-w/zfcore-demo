Detailed Description

Ограничения поиска при выборке из серверных таблиц. Аналог серверного класса TblRestr. Базовый абстрактный класс Объекты ограничения создаются с помощью метода zf::Core::createDataRestriction, т.к. за их генерацию отвечает драйвер
