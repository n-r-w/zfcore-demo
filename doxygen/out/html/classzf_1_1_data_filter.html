columnsКолонки набора данных
valuesЗначения для сравнения. Количество должно совпадать с количеством в columns
opsОператоры сравнения: [значение набора данных] [оператор] [value] Если не задано, то CompareOperator::Equal. Иначе количество должно совпадать с количеством в columns
rolesРоли для получения значения набора данных. Если не задано, то Qt::DisplayRole. Иначе количество должно совпадать с количеством в columns
optionsПараметры фильтрации. Если не задано, то NoOption. Иначе количество должно совпадать с количеством в columns
