Detailed Description

Реализует механизм последовательных обратных вызовов через встроенный таймер Необходим для синхронизации отложенной обработки, чтобы гарантировать соблюдение порядка отложенных вызовов и не полагаться на последовательность обработки тамеров Qt Кроме того реализована возможность приостановить обработку при необходимости вызвать критичный к обратным вызовам метод (например загрузка модели) Функция обработчик должна иметь вид: void sl_callbackManager(int key, const QVariant& data);.
