Detailed Description

Интерфейс плагина Плагин реализуется в модуле.

Отвечает за общую информацию о модуле и выступает в качестве фабрики для представлений, моделей и сервисов модуля

Асинхронное получение имени сущностей. В ответ необходимо послать сообщение MMEventEntityNamesMessage для sender. По умолчанию возвращает имя модуля для уникальных сущностей и грузит модель с данными для получения именя для остальных (медленно) Если имеется возможность получить расшифровку имени другими быстрыми способами (например через справочники) то надо переопределить этот метод

Parameters
requesterКому отправить ответное сообщение MMEventEntityNamesMessage
feedback_message_idИдентификатор сообщения, который надо передать в ответное MMEventEntityNamesMessage
entity_uidsИдентификаторы сущностей (все относятся к данному плагину)
name_propertyКакое свойство отвечают за хранение имени. Если какое свойство не валидно , то получить свойство с именем на основании zf::PropertyOption:FullName, а затем zf::PropertyOption:Name
languageЯзык. Если QLocale::AnyLanguage, то используется язык интерфейса
